# Hidden Numbers
We've recently obtained a counterfeit dollar bill printed by a nefarious organization. They aren't doing a very good job, so let's catch them before they can fix the issues.

We've determined that they are using a Xerox DocuColor 240 laser color printer. However, we'll need the printer's serial number before we can track them down or take any actions against them.

Submit the printer's serial number as `CTF{00000000}`
