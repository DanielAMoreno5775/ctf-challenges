# ctf-challenges

## File Structure

The repo is split into directories, each representing a different category.

Each category contains a separate directory for each challenge.

Each challenge has an `intended_solution.md` file which describes how to solve the challenge and provides the flag information. In addition, each challenge has a `public` directory which contains all of the files and information that will be provided to the player. If a player can download a file, that file will be found there. The `challenge_description.md` file will specify the challenge's name and message. In some cases, the file will also contain any hints that may be provided. Each hint has been placed in a different bullet point.
