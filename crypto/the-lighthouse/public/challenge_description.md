# Oracle of Delphi
Run this game on Linux, consult the oracle, and play the games.

Each game will randomly select World 0 or World 1. In the first 6 games, the Oracle of Delphi will ask for two messages and will select either Message 0 in World 0 or Message 1 in World 1. The message will be encrypted using the algorithm provided by the oracle. Games 7 through 9 also offer a decryption oracle to which you can pass a ciphertext, so long as the ciphertext was not returned by the encryption oracle.

Can you solve the games 100% of the time?
